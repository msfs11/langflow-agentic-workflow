# langflow-agentic-workflow

This repository implements agentic workflows using Langflow, a framework for building AI-powered applications. It includes custom components like the DataGridReader for processing CSV and XLSX files into data grids, and integrates with tools such as LibreChat for enhanced conversational AI capabilities.

## Features

- Custom components for data input and processing
- Agentic workflow configurations
- Docker-based deployment with docker-compose
- Integration with external services like LibreChat

## Getting Started

1. Clone the repository
2. Set up environment variables using `.env.example`
3. Run `docker-compose up` to start the services

For more details, refer to the Langflow documentation.
